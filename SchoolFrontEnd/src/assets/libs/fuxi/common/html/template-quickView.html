Template QuickPreview

Este template serve para exemplificar a utilização de um popover complexo, integrando um sistema de tabs. O seu tamanho pode ser personalizado aplicando uma combinação das classes abaixo listadas no template de construção do popover. Caso não se pretenda fazer override do tamanho default do popover não é necessário definir o template.

O header do quickview está definido como link para representar a possibilidade de abrir o conteúdo do quickview em página. Caso não se pretenda implementar este racional bastará não fazer uso da tag para o hyperlink e disponibilizar o nome da entidade diretamente no heading.
