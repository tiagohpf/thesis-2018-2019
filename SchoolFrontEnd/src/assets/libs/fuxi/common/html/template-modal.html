Template Modal

Este template serve para exemplificar a utilização de uma janela modal. O seu tamanho pode ser personalizado aplicando uma combinação das classes abaixo listadas.
Caso não se pretenda fazer override do tamanho default da modal não é necessário acrescentar a classe fx-modal-*definir-tamanho-aqui*.
A janela modal, independentemente da sua largura, tem uma altura máxima definida para a classe .modal-body. Essa altura máxima é de 550px. A partir daqui será gerado um scroll vertical.

A modal aberta pelo botão "preview default bootstrap modal" apresenta um exemplo no qual o título da modal está definido como link para representar a possibilidade de abrir o conteúdo da modal em página. Caso não se pretenda implementar este racional bastará não fazer uso da tag para o hyperlink e disponibilizar o nome da entidade diretamente no heading.
