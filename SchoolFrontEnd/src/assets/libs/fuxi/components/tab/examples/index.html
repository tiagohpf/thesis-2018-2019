fxTabs - AngularJS

(ver versão similar em Javascript)

Opções

Diretiva encarregada de encapsular as tabs
  • data-fxtab-refresh: Data attribute que ao alterar o seu valor permite refrescar as tabs
  • data-fxtab-data: Data attribute que recebe os dados a serem carregados
  • data-fx-onsortable: Data attribute que indica o nome da função a executar do lado da app, para ordenar os tabs.
    A função recebe dois parâmetros myFunction(fromIndex, toIndex).
  • data-fxtab-language: Data attribute que indica o ídioma em que são mostradas as mensagens
  • data-fx-statict-content: Data attribute que indica se o conteúdo das tabs é estático ou é carregado utilizando uma view (ex: ui-router)
Diretiva encarregada de renderizar as tabs
  • active: (true ou false) tab ativa
  • disable: (true ou false) tab pode ser clicável e colocada como ativa
  • dismissible: (true ou false) tab pode ser removida
  • fx-sortable: (true ou false) tab pode ser ordenável
  • href: Data attribute com o url da tab. É usado quando se clica com o botão direito do rato e se faz abrir link ou quando se faz o ctrl+click ou mouse middle click para abrir o url num novo tab.
  • status: Data attribute que dependendo do estado passado mostra icon a seguir ao titulo da tab (canto direito). Status: warning, error, success, info
  • data-fx-onselect: Data attribute que indica o nome da função a executar quando se activa uma tab
  • data-fx-ondeselect: Data attribute que indica o nome da função a executar quando se desactiva uma tab
  • data-fx-onremove: Data attribute que indica o nome da função a executar quando se remove uma tab
  • heading ou <tab-heading> : Texto do cabeçalho ou HTML
  • actions: (true ou false) indica se a tab tem um menu com acções ou não.
  • tabactions: lista de acções (fxtab-action) do menu.
Diretiva encarregue de renderizar as acções do menu das tabs
  • label: Label da acção do menu.
  • icon: Icon da acção do menu. (ex.: "{'glyphicon glyphicon-remove': true}") ou (ex.:"{'fuxicons fuxicons-thumb-tack': actionitem.show, 'fuxicons fuxicons-thumb-tack-o': !actionitem.show}")
  • show: (true ou false) indica se o menu item é visível ou não.
  • disabled: (true ou false) indica se o menu item está activo ou não.
  • permission: (true ou false) indica se tem permissões para aceder ao menu item ou não.
  • action: Acção que será usada no broadcast que é efectuado quando é pressionado o menu item. (ex.: 'removeTabAction')
    • actionitem="action" tabelm="elm"

How To? Step by Step

Verificar que tem incluído os seguintes ficheiros:
  • CSS
    • Bootstrap (/basepack/bootstrap-pt/3.0/css/bootstrap.min.css)
    • Fuxi (/fuxi/common/html/css/fuxi.css)
  • JS
    • jQuery (/basepack/jquery/1.9/jquery.min.js)
    • jQuery UI (/fuxi/lib/jqueryui/1.10/jquery-ui-1.10.4.custom.min.js)
    • Modernizr (/basepack/modernizr/2.6/modernizr.js)
    • Fuxi (/fuxi/common/exec/fuxi.js)
    • Bootstrap (/basepack/bootstrap-pt/3.0/js/bootstrap.min.js)
    • Angular JS (1.3.15) (https://ajax.googleapis.com/ajax/libs/angularjs/1.3.15/angular.min.js)
    • Angular UI-Router (//angular-ui.github.io/ui-router/release/angular-ui-router.js)
    • fxtab (/fuxi/components/tab/dist/fxtab.js)
Elemento DOM utilizado no Exemplo #1
<fxtabset data-fxtab-refresh="demo.reload" data-fxtab-data="demo.items" data-fx-onsortable="demo.sortable(fromIndex, toIndex)" data-fxtab-language="pt" fx-show-add="true" fx-add-callback="demo.addTab()"> <fxtab ng-repeat="tab in demo.items track by $id(tab)" active="tab.active" data-fx-onselect="demo.selected(tab)" data-fx-ondeselect="demo.deselected(tab)" data-fx-onremove="demo.removeTab($index)" disable="tab.disabled" actions="tab.actions"dismissible="tab.dismissible" data-fx-sortable="tab.sortable"> <fxtab-heading> <i class="glyphicon glyphicon-bell"></i> {{tab.title}} </fxtab-heading> </fxtab> </fxtabset> <div ui-view></div>
No ficheiro app.js pode verificar:
  • a forma de instanciar os módulos necessários para o correcto funcionamento do componente,
  • métodos de exemplo para o carregamento do conteúdo utilizando o UI-Router.
