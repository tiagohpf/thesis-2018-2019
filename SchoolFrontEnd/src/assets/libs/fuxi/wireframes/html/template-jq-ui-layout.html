How to

Este layout foi criado utilizando o plugin jQuery UI Layout.

O painel da esquerda (west), foi criado de forma a simular o splitter que até aqui temos vindo a adotar na FUXI. ver código

outerLayout.west.pane.find('.fx-sidebar-content').on('scroll', function () {
$this = $(this);
$prev = $this.prev();
if ($this.scrollTop() > 7 && !$this.hasClass("fx-sticky-shadow")) {
$prev.addClass('fx-sticky-shadow');
} else if ($this.scrollTop() <= 7 && $prev.hasClass("fx-sticky-shadow")) {
$prev.removeClass('fx-sticky-shadow');
} });

No painel central (center), foram criados dois paineis (north, south), no painel north, é onde consta o contéudo que está neste momento a ler.
O painel south tem informação dummy, onde o título do painel se encontra fixo.

No painel direito (east), está preenchido com informação dummy, mas de realçar que tanto o header como o footer se encontram fixos. Isto consegue-se ao definir a propriedade contentSelector do painel o selector correspondente à zona que pretendemos colocar scroll.
