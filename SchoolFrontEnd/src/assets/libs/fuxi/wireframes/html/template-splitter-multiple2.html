Título

Área de conteúdo da splitter bar

A área de botões ao fundo poderá ou não existir. Caso não exista o javascript de apoio a este template precisa ser editado.

How to

A instanciação do Splitter acenta na presença de um data atributo [data-toggle="splitter"] sobre um elemento do DOM.

Depois ao invocar a função de inicialização (FUXI.Splitter.init()), esta vai percorrer todos os elementos que contenham o data atributo definido anteriormente e assignar os eventos e funcionalidades definidas.

No elemento devemos definir os seguintes data atributos:

  • data-splitter-position: podemos definir o splitter nas seguintes posições: left, bottom e right
  • data-splitter-parent: indicar o identificador (#ID) do elemento "pai" do splitter
  • data-splitter-sidebar: indicar o identificador (#ID) do elemento sidebar, utilizado para expandir e collapsar.
  • data-splitter-content: indicar o identificador (#ID) do elemento que irá conter o conteúdo

No elemento podemos definir os seguites data atributos:

  • data-splitter-resize-max-width: indicar a largura máxima do splitter, default: 250
  • data-splitter-resize-min-width: indicar a largura mínima do splitter, default: 250
  • data-splitter-resize-max-height: indicar a altura máxima do splitter, default: 250
  • data-splitter-resize-min-height: indicar a altura mínima do splitter, default: 250

Exemplo da definição do painel da esquerda:

<div id="fx-splitter-bar" class="fx-splitter-bar" data-toggle="splitter"
data-splitter-position="left"
data-splitter-parent="#vertical-splitters"
data-splitter-sidebar="#fx-sidebar"
data-splitter-content="#fx-splitter-content"
data-splitter-resize-max-width="400"
data-splitter-resize-min-width="250"></div>

Para o splitter ter disponível a opção de redimensionamento é necessário estar incluída a biblioteca jQuery.UI
