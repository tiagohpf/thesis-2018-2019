FUXI - Framework User Experience Á Interface

version 1.20.0

Uma Framework Web que disponibiliza componentes e elementos UI implementados e/ou validados pela equipa de UX para uso transversal na Altice Labs.

Elementos UI

Snippets de elementos de user interface, tais como botões, formulários ou menus que poderão ser incluídos numa aplicação.

Componentes

Biblioteca de componentes desenvolvidas em jQuery para melhorar a interação do utilizador com as aplicações Altice Labs.

Componentes

Biblioteca de componentes desenvolvidas em Angular JS para melhorar a interação do utilizador com as aplicações Altice Labs.

Showroom

Exemplos de páginas onde são combinados os elementos UI com as componentes baseadas em JavaScript.

Um design transversal. Uma experiência única!

Utilizar a FUXI é mais do que garantir uma aparência similar para todos os produtos e serviços. É ter a garantia que, para além de se conferir um alinhamento corporativo em termos visuais, a experiência de utilização é a mais adequada às expetativas dos utilizadores.
