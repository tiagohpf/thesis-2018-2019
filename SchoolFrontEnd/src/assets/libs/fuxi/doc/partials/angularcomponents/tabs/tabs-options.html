Opções

O componente de Tabs disponibiliza as opções abaixo enumeradas, todas elas do tipo HTML data-*.

Os defaults são aplicados na ausência dos data-* no DOM.

Nome Tipo Default Descrição
Opções da diretiva fxtabset
data-fxtab-refresh texto '' Atributo que ao ser alterado permite refrescar as tabs.
data-fxtab-data texto '' Atributo que recebe os dados a serem carregados.
data-fx-onsortable texto '' Atributo que permite definir o nome da função a executar do lado da app.js para ordenar os tabs.
data-fxtab-language texto 'en' Atributo que permite definir a língua em que são mostradas as mensagens.
data-fx-statict-content texto '' Atributo que indica se o conteúdo das tabs é estático ou é carregado utilizando uma view (ex: ui-router).
fx-show-add texto false Atributo que indica se existe criação de tabs.
fx-add-callback texto '' Atributo que permite definir a função para tratar da criação de tabs. Depende da definição do atributo fx-show-add.
fx-add-tab-button-text texto '' Atributo que permite definir o texto do botão de criação de tab.
Opções da diretiva fxtab
active booleano false Atributo presente na tab ativa.
disable booleano false Atributo que indica se uma tab pode ser clicável e colocada como ativa.
dismissible booleano false Atributo que indica se a tab pode ser removida.
fx-sortable booleano true Atributo que indica se a tab pode ser reposicionada.
status HTML data-* '' Atributo que permite mostrar um ícone após o título da tab indicado o estado desta. Poderá ser: warning | error | success | info.
href HTML data-* '' Atributo que permite definir o url da tab. É usado quando se clica com o botão direito do rato e se faz abrir link ou quando se faz o ctrl+click ou mouse middle click para abrir o url num novo tab.
data-fx-onselect HTML data-* '' Atributo que permite indicar o nome da função a executar quando se activa uma tab.
data-fx-ondeselect HTML data-* '' Atributo que permite indicar o nome da função a executar quando se desativa uma tab.
data-fx-onremove HTML data-* - Atributo que permite indicar o nome da função a executar quando se remove uma tab.
heading HTML data-* - Atributo que permite definir o texto do cabeçalho.
