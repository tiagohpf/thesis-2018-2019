Eventos e Métodos

O selectboxtree disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
selectBoxTree::formSubmitted Executado quando o form é submetido (configurável).

Métodos

O serviço fxselectboxtreeService exposto pela diretiva fxselectboxtree permite:

setCurrentLanguage (newCurrentLanguage)

Método que permite definir a língua do componente.

getCurrentLanguage ()

Método que permite obter a língua do componente.

loadCustomLocales (customLocales)

Método que permite o carregamento de novas línguas. Estas devem respeitar a estrutura { 'lang' : { key: value, ...} }

As keys podem ser:

  • PLACEHOLDER-SEARCH
  • PLACEHOLDER-ELEMENT
  • MESSAGE-EMPTY-DATA
  • MESSAGE-LOADING
  • REQUEST-ERROR-MSG
  • REQUEST-EMPTY-DATA
  • REQUEST-SEARCHING
  • REQUEST-INITIAL-SEARCH
  • CHARACTER-PLURAL-SUFFIX
setLabelByKey (key, text)

Método que permite modificar uma key da língua definida.

getLocale (key, lng)

Método que permite obter uma key de uma determinada língua.
