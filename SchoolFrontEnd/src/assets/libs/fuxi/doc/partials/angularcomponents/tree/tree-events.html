Eventos e Métodos

A tree disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
treeCtrl.treeAfterLoad Lançado quando a tree termina de ser carregada.
treeCtrl.treeElementClicked Lançado quando um elemento da árvore é clicado.
treeCtrl.onRequestSuccess Lançado quando ocorre um pedido com sucesso.
treeCtrl.onRequestError Lançado quando ocorre um pedido sem sucesso.

Métodos

O serviço treeServices exposto pela diretiva fxTree permite:

prepareJSONData (data, additionalPlugins, additionalPluginConfigurations, additionalCoreConfigurations)

Método que prepara um objecto com as configurações e dados necessários para o correcto carregamento da jsTree.

getTree (id)

Método que retorna o objecto jsTree.

deleteTree (id)

Método que destrói o objecto jstree.

storeNewTree (id, tree, treeInfo)

Método que permite guardar a nova tree.

updateTree (id, tree, treeInfo)

Método que atualiza os dados da tree.

existsTree (id)

Método que verifica se a tree existe.

A diretiva apresenta 3 formas de inicialização:

callInitializeTreeWithJSON(data, additionalPlugins, additionalPluginConfigurations, additionalCoreConfigurations)

Método que carrega a tree utilizando os nodos passados como parâmetro no método.

callInitializeTreeWithRest(additionalPlugins, additionalPluginConfigurations, additionalCoreConfigurations)

Usando o endereço definido em data-tree-load-url, este método carrega os dados usando o módulo $http.get() do AngularJS.

callInitializeTreeEmpty(additionalPlugins, additionalPluginConfigurations, additionalCoreConfigurations)

Método que permite instanciar a tree sem nodos.

A diretiva permite ainda fazer override aos seguintes métodos:

ref.treeElementClicked = function (ev, data) { };

Método invocado quando um elemento da árvore é clicado.

ref.treeAfterLoad = function (tree) { };

Método invocado após carregamento dos nodos da árvore.

ref.onRequestSuccess = function (tree, data) { };

Método invocado quando ocorre um pedido com sucesso.

ref.onRequestError = function (data, status) { };

Método invocado quando ocorre um pedido sem sucesso.
