Opções

A tree disponibiliza as opções abaixo enumeradas, todas elas do tipo HTML data-*.

Os defauts são aplicados na ausência dos data-* no DOM.

Nome Tipo Default Descrição
data-fx-tree HTML data-* - Diretiva encarregada de dar o comportamento desejado ao elemento DOM.
data-fx-inst-tree HTML data-* - Diretiva usada para a inicialização (3 modos).
data-tree-load-url texto - Indica o caminho para o ficheiro com os dados a serem carregados na tree.
