Opções

O selectboxtree disponibiliza as opções abaixo enumeradas, todas elas do tipo HTML data-*.

Os defauts são aplicados na ausência dos data-* no DOM.

Nome Tipo Default Descrição
data-fx-select-box-tree HTML data-* - Diretiva encarregada de dar o comportamento desejado ao elemento DOM.
data-select-box-tree-data texto '' Permite receber os dados a serem carregados na JSTREE. É obrigatório sobre o atributo data-fx-select-box-tree.
data-select-box-tree-url texto '' Indica o caminho para o ficheiro com os dados a serem carregados na JSTREE. O carregamento dos dados utilizando este método permite que os mesmos sejam carregados aquando a instanciação da directiva ou então quando clicamos sobre a dropdown para expandir e visualizar os dados da JSTRE.
data-select-box-tree-demand-mode texto onInstantiation Permite escolher o modo de carregamento dos dados:
  • onInstantiation - carregamento na instanciação da diretiva;
  • onDemand - carregamento na primeira abertura da dropdown.
data-select-box-tree-selected-values array [] Permite receber os items previamente selecionados.
data-select-box-tree-multi-values HTML data-* Ao colocar este data atributo ativa a multi-seleção de items. Indica que se pretende multi-seleção de items.
data-select-box-tree-full-width HTML data-* Ao colocar este data atributo o componente passa a ocupar 100%. Indica se o componente deve ocupar 100% da largura do seu elemento pai.
data-select-box-tree-language texto 'en' Indica o ídioma em que são mostradas as mensagens.
data-fx-select-box-tree-target HTML data-* Opcional Directiva (aplicada ao input) encarregada de dar o comportamento de obrigatoriedade ao elemento DOM.
data-select-box-tree-remote-search HTML data-* Ao colocar este atributo o componente utiliza pesquisa remota. Indica se o componente utiliza pesquisa remota.
data-select-box-tree-url-params texto 'qs' Permite passar parâmetros extra ao pedido de pesquisa remota. Definição da key que irá passar o valor de pesquisa.
data-select-box-tree-full-label HTML data-* Ao colocar este atributo indica que pretende obter uma string com todos os valores selecionados Permite indicar se é para obter uma string com todos os valores selecionados na tree, utiliza o valor do parser definido no atributo data-select-box-tree-full-label-parser.
data-select-box-tree-full-label-parser texto ' , ' Permite passar a string a utilizar quando se retorna todos os valores selecionados.
data-select-box-tree-write-interval numérico '300' Permite definir o intervalo de tempo (ms) entre escrita até fazer o pedido para a busca remota.
data-select-box-tree-submitted-event-name texto 'selectBoxTree::formSubmitted' Nome do evento a executar no caso do form ser submitido.
data-select-box-tree-read-only HTML data-* Ao colocar este atributo o componente abre em modo de leitura Permite indicar se é para carregar o componente em modo apenas de leitura.
data-select-box-tree-disabled-browser-contextmenu HTML data-* false Se definido a true permite desabilitar o context menu dos browsers ao fazer right-click sobre um nó da árvore.
