Eventos e Métodos

As tabs disponibilizam os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
fxtabAction::_action_ Executado quando o ocorre a _action_ do menu de opções.

Métodos

O serviço fxtabService exposto pela diretiva fxtabs permite:

setCurrentLanguage (newCurrentLanguage)

Método que permite definir a língua do componente.

getCurrentLanguage ()

Método que permite obter a língua do componente.

loadCustomLocales (customLocales)

Método que permite o carregamento de novas línguas. Estas devem respeitar a estrutura { 'lang' : { key: value, ...} }

As keys podem ser:

  • PLACEHOLDER-SEARCH
  • LABEL-VIEW-CONTENT-LEFT
  • LABEL-VIEW-CONTENT-RIGHT
  • LABEL-ALL-TABS
setLabelByKey (key, text)

Método que permite modificar uma key da língua definida.

getLocale (key, lng)

Método que permite obter uma key de uma determinada língua.
