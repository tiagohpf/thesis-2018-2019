Opções

O editor de expressões disponibiliza as opções abaixo enumeradas.

Nome Tipo Default Descrição
layoutOptions Objeto ver código
Opções do jQuery UI Layout.
hideWestPanel booleano false Controla a visibilidade do painel da esquerda.
hideEastPanel booleano true Controla a visibilidade do painel da direita.
clientCheckRuleSyntaxValidator function undefined Função utilizada para verificar a sintaxe da regra. Por defeito está a undefined e cada equipa deverá implementar a sua lógica.
expressionItems Array [] Serve para popular a árvore das expressões. Admite JSON no formato permitido pela jsTree.
operatorItems Array [] Serve para popular a área de operadores. Admite JSON com a estrutura exemplificada. ver código
templateGroup HTML ver código
Template HTML para o grupo de ações.
templateGroupItem HTML ver código
Template HTML para os items do grupo de acções.
templateAllActions HTML ver código
Template HTML para todas as acções definidas.
languageStrings Objeto ver código
Objeto com as mensagens utilizadas.
