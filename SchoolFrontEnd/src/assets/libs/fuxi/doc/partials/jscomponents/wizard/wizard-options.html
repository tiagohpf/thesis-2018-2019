Opções

O wizard disponibiliza as opções abaixo enumeradas.

Nome Tipo Default Descrição
isAsync booleano false Define se o comportamento é executado quando se instancia ou no evento window.load. False faz com que ocorra no window.load.
mode string FUXI.Wizard.modeEnum.INSERT Define o modo como o wizard é disponibilizado ao utilizador. INSERT | EDIT | VIEW. Pode optar por passar o data-attribute em vez de declarar esta opção na zona de scripting. Consulte modos de ver um wizard para perceber o que cada modo representa.
validateOnPrevious booleano false Define se a validação deve ser disparada quando se retrocede no wizard. Pode optar por passar o data-attribute em vez de declarar esta opção na zona de scripting.
gotoStopOnError booleano false Define se a validação deve apenas validar o passo atual ou todos os passos até ao passo de destino.
