Visibilidade das colunas

O DataTables possui um extra que disponibiliza uma forma de mostrar ou esconder colunas da tabela. Contudo a FUXI não empacota este extra pois tipicamente a opção de gerir a visibilidade das colunas é colocada numa toolbar com mais opções.

Ainda assim, caso pretenda usar este extra, em DataTables ColVis pode encontrar informação detalhada.

O exemplo abaixo demonstra a forma que a FUXI sugere para gerir a visibilidade das colunas. A codificação será responsabilidade das esquipas de produto.
