Eventos e Métodos

A dockbar disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Todos os eventos deverão ser associados a um objeto dockbar. Um objeto dockbar é identificado pelo data-attribute data-menu="offscreen-menu". Exemplo: $('div[data-menu="offscreen-menu"]').on('toggled.offscreenmenu.fuxi', function (event) { });.

Evento Descrição
toggled.offscreenmenu.fuxi Executado quando se clica num botão localizado dentro do elemento DOM com a classfx-offscreen-menu. No event é passado a propriedade state que admite os seguintes estados:
  • collapsed: quando a dockbar se encontra colapsada.
  • expanded: quando a dockbar se encontra expandida.
attach.offscreenmenu.fuxi Executado quando se clica num dos botões identificados pelo data-attribute data-menu="detach" ou data-menu="reattach". No event é passado a propriedade state que admite os seguintes estados:
  • detach: quando a dockbar se encontra destacada.
  • reattach: quando a dockbar se encontra fixa.

Métodos

FUXI.OffscreenMenu.init()

Método que inicializa a dockbar.
