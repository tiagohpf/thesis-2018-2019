Selectboxtree

A versão JavaScript do Selectboxtree assenta em duas componentes.

  • Para desenhar a árvore utiliza-se a biblioteca jsTree. Em jsTree - jQuery tree plugin pode consultar documentação mais pormenorizada.
  • Outros elementos são disponibilizados pelo bootstrap, nomeadamente classes do formulário ou a janela modal onde poderá ser instanciado. Em bootstrap pode consultar documentação mais pormenorizada.

Caso se pretenda uma versão AngularJS, consultar em versão AngularJS a documentação.
