Eventos e Métodos

O widget select disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
shown.fx.widgetselect Executado quando é mostrado o widget select.
addsave.fx.widgetselect Executado quando é criado um novo item. O item deve ser criado neste evento e após ser criado com sucesso devem ser invocados os métodos handleShowSelectView e drawItem.
addcancel.fx.widgetselect Executado quando é cancelada a criação de um novo item.
selectconfirm.fx.widgetselect Executado quando são seleccionados os items. Retorna no argumento os items selecionados. Fecha a modal com o widget select.
selectcancel.fx.widgetselect Executado quando é cancelada a selecção dos items. É feito o reset dos items que possam ter sido eliminados ou selecionados. Fecha a modal com o widget select.
showadd.fx.widgetselect Executado quando é mostrada a view de inserção de novo item.
showselect.fx.widgetselect Executado quando é mostrada a view de seleção.
itempreview.fx.widgetselect Executado quando é mostrada a pré-visualização do item. É aqui que se deve ir buscar os dados de detalhe do item e depois de os carregar para o DOM deve ser invocado o método layout.resizeAll().
itemremove.fx.widgetselect Executado quando é removido um item.
removeall.fx.widgetselect Executado quando são removidos todos os itens.
sortby.fx.widgetselect Executado quando, em modo grid, se despoleta uma ordenação sobre uma das colunas da tabela.
search.fx.widgetselect Executado quando se clica sobre a lupa para efetuar uma pesquisa.

Métodos

itemsList(itemsToShow, itemsCount)

Método que recebe a lista de itens a serem mostrados. Necessita o número total de itens.

selectedItemsList(itemsSelected)

Método que recebe a lista de itens pré-selecionados.

reset()

Método que faz reset a tudo. Executa o método resetList().

resetList()

Método que limpa a lista dos elementos disponíveis.
