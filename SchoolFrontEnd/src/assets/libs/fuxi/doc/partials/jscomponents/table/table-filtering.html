Filtering

Uma tabela DataTable permite fazer filtering da informação das colunas. Em DataTables filtering pode encontrar informação mais detalhada. O exemplo abaixo mostra uma tabela que permite filtragem em todas as colunas.

O Basepack disponibiliza um plug-in que permite colocar o input de filtragem imediatamente abaixo do título da coluna. Para isso deverá incluir o ficheiro jquery.dataTables.columnFilter.js disponível em src="/basepack/datatables-pt/1.9/js/jquery.dataTables.columnFilter.js".

Atenção: Este plug-in apesar de listado no site do DataTables não é um plugin oficial pelo que poderão ser necessários desenvolvimentos adicionais por parte das equipas de produto.

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.
