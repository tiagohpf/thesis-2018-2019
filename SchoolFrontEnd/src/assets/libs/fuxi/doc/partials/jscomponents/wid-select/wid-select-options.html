Opções

O widget select disponibiliza as opções abaixo enumeradas.

Nome Tipo Default Descrição
openMode string modal Define o modo de apresentação do widget select. modal | inline
showMode string grid Define o modo como a lista de elementos é apresentada. grid | tree
layoutOptions Objeto ver código
Opções do jQuery UI Layout.
showAddView booleano true Controla a visibilidade da opção de criar.
showPreviewView booleano true Controla a visibilidade da opção de prévisualização.
selectItemMandatory booleano false Controla a obrigatoriedade, ou não, da seleção de pelo menos um elemento.
selectItemMandatoryMessage string '' Permite disponibilizar uma mensagem no caso de não se selecionar nenhum elemento da lista.
multiSelection booleano true Permite, ou não, selecionar mais do que um elemento.
showPageOptionSelectAll booleano false Controla a visibilidade da opção de selecionar todos os items da listagem que está a ser visualizada.
itemsPerPage número 20 Define o número de elementos apresentados por página.
allowDrag booleano true Permite arrastar, ou não, a modal.
returnSelectedItemsSorted booleano false Permite que os items retornados sejam, ou não, ordenados.
returnSelectedItemsHtml booleano false Permite, ou não, retornar os items em HTML.
itemsTemplate HTML ver código
Template HTML para cada item da lista. Apenas aplicável quando o modo selecionado é grid.
selectedItemTemplate HTML ver código
Template HTML para cada elemento selecionado da lista.
selectedItemReturnTemplate HTML '' Template HTML para cada elemento seleccionado retornado da lista.
jstreePlugins array [] Quando em modo tree permite disponibilizar plugins à jsTree. Em jsTree - jQuery tree plugin pode consultar documentação mais pormenorizada.
jstreePluginsConfiguration array [] Quando em modo tree permite passar dados de configuração aos plugins adicionados à jsTree.
jstreeCoreConfiguration array [] Quando em modo tree permite passar dados de configuração à jsTree.
clientAddValidator function undefined Caso se disponibilize a opção de criar, o programador poderá criar uma função de validação para a informação a inserir.
notificationCallback function undefined Caso se disponibilize a opção de criar, o programador poderá criar uma função para notificar o utilizador se o elemento foi criado, ou não, com sucesso.
lazyLoadCallback function undefined Nome da função a chamar caso o programador decida implementar a técnica de lazy load para carregar os dados da listagem. Esta função é da responsabilidade do programador. Recebe o número da página actual.
