RowReorder

O DataTables possui um plug-in que disponibiliza uma forma de reordenar linhas da tabela. Contudo a FUXI não empacota este plug-in pois ele apresenta algumas limitações, nomeadamente quando usado em conjunto com o ColReorder.

Ainda assim, caso pretenda usar este extra, em DataTables RowReorder pode encontrar informação detalhada. Pode igualmente fazer upgrade para a versão 1.10 do DataTables que já possui um plug-in oficial de RowReorder.

O exemplo abaixo demonstra a forma que a FUXI sugere para apresentar uma linha como reordenável. A codificação será responsabilidade das esquipas de produto. Dependendo da forma que a equipa de produto desenvolva a funcionalidade sugere-se que seja aplicada uma opacity: .5 ao elemento arrastável enquanto o mesmo se encontra a ser arrastado.

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.

Para que o ícone representativo da funcionalidade de arrastar seja mostrado deve adicionar a class .fx-allow-row-reorder ao elemento tbody.
