Sumário de um wizard step

Este snippet demonstra a forma como é mostrado ao utilizador um sumário das escolhas efetuadas nos steps. Caso a escolha do utilizador tenha impacto no caminho do wizard a FUXI coloca um ícone diferente no sumário. Para que este ícone apareça, no sumário, é necessário definir a true a propriedade stepDecision.

Tenha por favor em atenção que este sumário deve ser usado com moderação, apresentando ao utilizador apenas informação relevante.
