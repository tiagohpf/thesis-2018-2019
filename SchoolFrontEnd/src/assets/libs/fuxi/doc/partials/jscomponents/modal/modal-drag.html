Arrastar modal

Por vezes pode ser importante permitir que uma janela modal seja arrastável pelo ecrã. Para permitir essa funcionalidade recorre-se a uma biblioteca externa. Em jquery UI Draggable pode consultar documentação mais pormenorizada.

O snippet de código abaixo demonstra uma instanciação simples de uma janela arrastável. A modal arrastável não deve apresentar fundo. Como tal deve ser instanciada com data-backdrop="false" e data-keyboard="false".
