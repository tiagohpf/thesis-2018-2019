Loading sobre a página

Este snippet demonstra como aplicar um bloco de loading à página.

Os valores por defeito aplicados ao BlockUI são disponibilizados através da função FUXI.BlockUiDefaults.init();. No entanto, quando se trata de um loading aplicado à página são sobrepostos alguns desses valores.

Para efeitos de demo utilizamos um botão para fazer trigger ao loading da página. Pode clicar em qualquer área do documento para remover o estado de loading.
