Widget select

O widget select assenta em diversas componentes:

  • Para a construção dos painéis utiliza-se a biblioteca UILayout. Em jQuery UI Layout pode consultar documentação mais pormenorizada.
  • Para desenhar a árvore utiliza-se a biblioteca jsTree. Em jsTree - jQuery tree plugin pode consultar documentação mais pormenorizada.
  • Diversos elementos são disponibilizados pelo bootstrap, nomeadamente botões e a janela modal, caso se opte por disponibilizar o widget select numa janela modal. Em bootstrap pode consultar documentação mais pormenorizada.
