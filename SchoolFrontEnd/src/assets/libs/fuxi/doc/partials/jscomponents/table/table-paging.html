Paging

Existem diversos mecanismos para fazer a paginação de uma tabela do tipo DataTable. Em DataTables Custom pagination types pode encontrar informação mais detalhada dos plug-ins de paginação disponíveis.

A FUXI inclui paginação do tipo input, que consiste em disponibilizar os elementos abaixo referidos.

  • Botão para voltar ao início da listagem
  • Botão para retroceder uma página
  • Botão para avançar uma página
  • Botão para ir para o fim da lsitagem
  • Input para ir para uma determinada página

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.

Através da utilização da função FUXI.Presets.DataTables já será definido o tipo de paginação como input. Contudo, caso não use os presets, deverá definir a propriedade sPaginationType da seguinte forma: 'sPaginationType': 'input'.
