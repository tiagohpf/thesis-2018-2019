Rowgroup

O DataTables possui um extra que disponibiliza uma forma de agrupar linhas. Para o utilizar deve incluir o ficheiro jquery.dataTables.rowGrouping.min.js

Em DataTables Row Grouping Example pode encontrar informação útil para este plug-in.

O exemplo abaixo demonstra o aspeto das linhas agrupadoras.

Atenção: Este plug-in apesar de listado no site do DataTables não é um plugin oficial pelo que poderão ser necessários desenvolvimentos adicionais por parte das equipas de produto.

A FUXI apenas disponibiliza estilos para a linha agrupadora. O desenvolvimento deverá ser assegurado por parte das equipas de produto.

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.
