Data attributes da dockbar

Para que uma dockbar funcione corretamente é necessário definir uma série de data-attributes.

Datta-attribute Descrição
data-menu="offscreen-menu" Declarado no elemento que encapsula toda a dockbar. Declara o objeto como dockbar.
data-menu="detach" Caso se pretenda usar a funcionalidade de destacar deve ser declarado no botão usado para o efeito.
data-menu="reattach" Caso se pretenda usar a funcionalidade de fixar deve ser declarado no botão usado para o efeito.
