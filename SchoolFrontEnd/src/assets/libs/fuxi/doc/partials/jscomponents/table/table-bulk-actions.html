Ações em bulk sobre a tabela

O exemplo abaixo apresenta como disponibilizar ações em bulk sobre uma tabela. A FUXI sugere que do lado esquerdo sejam colocadas as operações que afetem diretamente as linhas e do lado direito as que afetem a tabela como um todo.

Cada um dos botões que afetem as linhas apenas deverão ficar ativos se pelo menos uma linha da tabela estiver selecionada.

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.

Atenção: O código abaixo é meramente exemplificativo como forma de demonstrar a ativação/desativação dos botões e o aspeto de uma linha selecionada. A implementação da funcionalidade é responsabilidade das equipas de produto. Ao selecionar uma checkbox deve garantir-se que a class .disabled é removida do botão e que ao respetivo tr é aplicada a class .selected.
