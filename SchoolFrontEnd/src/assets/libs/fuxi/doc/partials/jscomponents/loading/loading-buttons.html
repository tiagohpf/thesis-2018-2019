Loading em botões

Este snippet demonstra como aplicar um bloco de loading a um botão e tira partido dos métodos disponibilizados pelo bootstrap para os botões. Em bootstrap button methods pode consultar documentação mais pormenorizada.

Para efeitos de demo ao fim de 5 segundos o loading terminará.

Tenha por favor em atenção que se usar a biblioteca jQueryUI esta deve ser instanciada antes do ficheiro JavaScript do bootstrap.
