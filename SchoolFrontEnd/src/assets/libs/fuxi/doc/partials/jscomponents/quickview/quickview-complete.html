Quickview complexo

Este snippet demonstra aquilo que a FUXI denomina um quickview complexo, isto é, um quickview com cabeçalho, conteúdo e rodapé. Para este caso, como pretendemos injetar algumas classes FUXI será necessário definir via JavaScript um template.

O título do quickview está definido como link para representar a possibilidade de abrir o conteúdo do quickview em página. Caso não se pretenda implementar este racional bastará não fazer uso da tag para o hyperlink e disponibilizar o nome da entidade diretamente no heading.
