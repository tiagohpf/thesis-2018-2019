Tabela - overview

O exemplo abaixo apresenta o exemplo mais simples de uma tabela.

Para instanciar uma tabela como a demonstrada abaixo deve fazer uso dos seguintes ficheiros:

  • Para a tabela propriamente dita: jquery.dataTables.css e jquery.dataTables.min.js que são disponibilizados via basepack.
  • Para a paginação: inputPaging.min.js disponibilizado pela FUXI.
  • Através das funções FUXI.Presets.DataTables e FUXI.Utils.SetDTPagination são garantidas algumas regras de base para as tabelas.
