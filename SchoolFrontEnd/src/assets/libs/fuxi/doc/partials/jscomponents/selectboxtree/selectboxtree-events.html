Eventos e Métodos

O selectboxtree disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
deselecteditem.fx.selectboxtree Executado quando se remove um item selecionado.
selecteditem.fx.selectboxtree Executado quando se selecciona um item.

Métodos

getSelectedValues()

Método que retorna a lista de items selecionados

getSelectedIDs()

Método que retorna string com os IDs selecionados, separados pelo separador indicado nas opções (selectedIDsSpacer).

openDropdown()

Método que permite abrir o dropdown.

closeDropdown()

Método que permite fechar o dropdown.

triggerDropdown()

Método que permite executar o close ou open do dropdown, dependendo do estado em que este se encontre.

resetSelectedItems()

Método que permite repor os items selecionados no seu estado inicial.

clearAll()

Método que permite limpar todos os items selecionados.
