Loading sobre um objeto

Este snippet demonstra como aplicar um bloco de loading a um objeto, no caso a uma select, mas pode ser aplicado a qualquer elemento presente no DOM.

Os valores por defeito aplicados ao BlockUI são disponibilizados através da função FUXI.BlockUiDefaults.init();.

Para efeitos de demo utilizamos um botão para fazer trigger ao loading sobre o objeto. Ao fim de 10 segundos o loading terminará.
