Exportar tabela

O DataTables possui um extra que disponibiliza formas para exportar a informação de uma tabela. Contudo a FUXI não empacota este extra pois tipicamente as opções de exportação são colocadas numa toolbar com mais opções.

Ainda assim, caso pretenda usar este extra, em DataTables Table Tools pode encontrar informação detalhada.

A FUXI sugere duas formas para exportar a informação da tabela, uma standard e outra avançada. A codificação será responsabilidade das equipas de produto. No exemplo abaixo estão demonstradas ambas as formas. Contudo, sobre a mesma tabela, apenas deverá existir uma.
