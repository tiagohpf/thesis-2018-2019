Progresso mensurável

Este snippet demonstra como instanciar uma barra de progresso mensurável.

No snippet abaixo está exemplificada uma barra de progresso default. Podem instanciar-se outros tipos concatenando as classes contextuais progress-bar-success, progress-bar-info, progress-bar-warning ou progress-bar-danger à classe progress-bar.

Caso não pretenda que o valor percentual do carregamento seja disponibilizado basta acrescentar a class .sr-only à label que se encontra dentro da barra de progresso.
