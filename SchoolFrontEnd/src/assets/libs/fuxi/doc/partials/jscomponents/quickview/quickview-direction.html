Orientação do quickview

Este snippet demonstra como instanciar um quickview simples nas quatro direções possíveis: cima, direita, baixo, esquerda.

O snippet faz uso dos data-attributes mas pode-se chegar ao mesmo resultado instanciando o objeto em JavaScript.

Para além de definir o eixo para onde se pretende abrir o quickview é boa política atribuír também o valor auto de forma a garantir que se não houver espaço para a abertura do quickview no eixo definido ele inverte a direção da abertura.

Na versão atual do bootstrap disponibilizada no basepack existe um erro com o data-placement="bottom auto".

Usando este valor no atributo data-placement o quickview abre sempre para cima. Para abrir para baixo retirar o auto, tendo atenção às implicações desta alteração.
