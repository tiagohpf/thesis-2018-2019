Opções

O Selectboxtree disponibiliza as opções abaixo enumeradas.

Nome Tipo Default Descrição
required booleano false Altera layout para indicar que o componente é de preenchimento obrigatório.
multiple booleano true Permite multi seleção de items.
disabled booleano true Mostra o componente sem a possibilidade de selecionar items.
showMode texto 'tree' Permite escolher o modo de visualização dos items - grid | tree.
showSelectAll booleano false Mostra a opção de selecionar todos os items.
showSearch booleano true Mostra a opção de pesquisar sobre os items.
numberDisplayed número 0 Numero até ao qual são mostrados os items. Após este número é apresentada uma tag com o numero de items selecionados seguido do texto definido na opção nSelectedText.
nSelectedText texto '' Texto a ser apresentado quando a opção numberDisplayed está definida.
allSelectedText texto '' Texto a ser apresentado quando se tem todos os items selecionados.
fullWidth booleano true Renderiza o componente ocupando 100% da largura.
searchPlaceholder texto 'pesquisar...' Placeholder a ser mostrado no campo de pesquisa.
selectedPlaceholder texto 'selecione...' Placeholder a ser mostrado no campo dos items selecionados.
selectableName texto 'isgroup' Nome da propriedade no JSON utilizada para indicar se o item é um agrupador e não deve fazer parte da lista de items selecionados. Ao ter um item da árvore definido a true nesta propriedade é utilizado o plugin "changed" que se encontra no ficheiro jstree.misc.js, pelo que deve estar incluído na página.
selectedIDsSpacer texto ',' Caractere utilizado pela função getSelectedIDs() na separação dos identificadores dos items retornados.
items array [] Lista de items a carregar na árvore (showMode: 'tree'). Estrutura do JSON de acordo com a documentação do jsTree.
selectedItems array [] Lista de items pré selecionados. Estrutura do elemento [{"id": "ID-VALUE", "text": "TEXT-VALUE"}]
jstreePlugins array ver código
Lista de plugins usados pelo jsTree (showMode: 'tree').
jstreePluginsConfiguration array ver código
Lista de configurações dos plugins usados pelo jsTree (showMode: 'tree').
jstreeCoreConfiguration array ver código
Lista de configurações base utilizadas pelo jsTree (showMode: 'tree').
selectItemTemplate html ver código
Template HTML para cada item seleccionado.
gridItemTemplate html ver código
Template HTML para cada item representado na grelha (showMode: 'grid').
