Eventos e Métodos

O editor de expressões disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Evento Descrição
loaded.ee Executado quando o editor de expressões é carregado.
confirmclick.ee Executado quando o botão com a class .fx-ee-confirm é clicado. Este botão tipicamente tem a designação aplicar.
cancelclick.ee Executado quando o botão com a class .fx-ee-cancel é clicado. Este botão tipicamente tem a designação cancelar.
addexpression.ee Executado quando se adiciona uma regra a partir da árvore. A adição pode ser feita através de duplo clique ou arrastando o item para a textarea.
clearrule.ee Executado quando o botão com a class .fx-clear-rule é clicado. Este botão tipicamente tem a designação limpar.
checksyntax.ee Executado quando o botão com a class .fx-check-syntax é clicado. Este botão tipicamente tem a designação verificar sintaxe.

Para efeitos de demo, ao escrever uma regra e clicar no botão para verificar sintaxe, será gerado um número aleatório (par ou ímpar). Mediante esse número gerado será mostrado feedback de erro ou sucesso.
operatorclick.ee Executado quando qualquer botão com a class .fx-ee-btn-action é clicado. Estes botões dizem respeito aos operadores disponibilizados.
searchexpressionclick.ee Executado quando se efetua uma pesquisa na árvore que contém as expressões.
searchactionclick.ee Executado quando se efetua uma pesquisa na lista dos operadores.

Métodos

destroy()

Método que elimina a regra e todos os eventos associados ao elemento.
