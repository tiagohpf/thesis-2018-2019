Instanciação de um splitter

O markup do splitter deverá ser explicitamente disponibilizado na página.

O snippet de código abaixo ilustra os ficheiros necessários ao correto funcionamento do splitter. Demonstra ainda a instanciação dos diferentes elementos usados. Tenha por favor em atenção que para o splitter ser corretamente visualizado será necessário definir uma altura fixa para o parent do elemento com a class .fx-uilayout-container.

A FUXI sugere que quando se colapsa uma sidebar seja feito o append do(s) ícone(s) representativo(s) da(s) funcionalidade(s) disponibilizada(s) na sidebar.

pesquisar

Área de conteúdo da splitter bar.

Tipicamente é utilizada para pesquisas, mas pode conter qualquer tipo de conteúdo.

A área de botões ao fundo poderá ou não existir.

Splitter

A componente Splitter consiste num conjunto de paineis que podem ser colapsados ou arrastados. Podemos definir paineis à esquerda, direita ou abaixo. Nesta página de demo apresentamos o exemplo mais simples, que consiste numa sidebar e numa área de conteúdo.
