Scrollable tabs

Por vezes existe a necessidade de disponibilizar um sistema de tabs composto por bastantes elementos. Para fazer face a esse cenário disponibiliza-se um mecanismo de slider de forma a permitir encaixar quantos tabs se desejar sem aumentar o espaço vertical consumido.

Instanciação

O markup da componente Scrollable tabs deverá ser explicitamente disponibilizado na página.

O snippet de código abaixo ilustra os ficheiros necessários ao correto funcionamento da componente. Demonstra ainda de forma muito resumida a sua instanciação. Opções

Nome Tipo Default Descrição
isAsync booleano false Define se o comportamento é executado quando se instancia ou no evento window.load. False faz com que ocorra no window.load.
animateDuration número | string 400 Define a duração da animação em milisegundos ou através de algumas constantes. Quanto maior o valor indicado mais lenta será a animação. As constantes poderão ser: fast (200ms) ou slow (600ms)
fixedIncrement número 80 Define o incremento que será dado cada vez que se navega na seta de navegação.
tabTemplate HTML ver código
Template de um tab.
tabDismissibleTemplate HTML ver código
Template para o elemento que permite fazer dismiss a um tab. Caso se defina um template de ações para um botão, a opção de dismiss deverá ser colocada dentro dessa área de ações.
tabStatusTemplate HTML ver código
Template para indicar se existe algum estado de validação subjacente ao tab.
tabActionsTemplate HTML ver código
Template para colocar ações dentro de um tab que afetem o mesmo tab.
tabDropdownTemplate HTML ver código
Template para cada uma das opções colocadas dentro do dropdown menu que permite ver a informação do tab.

Eventos

Evento Descrição
selected.fx.slider.tab Executado quando se seleciona uma tab.
moved.fx.slider.tab Executado quando se move uma tab
added.fx.slider.tab Executado quando se adiciona uma nova tab.
removed.fx.slider.tab Executado quando se remove uma tab.
removedAll.fx.slider.tab Executado quando se removem todas as tabs.

right.fx.slider.tab Executado quando se clica sobre o botão que permite mover os tabs para a direita.

Métodos

tabContentLoaded(tab, status, title)

Método que remove o ícone de loading e coloca um ícone de status e respetivo title, caso seja definido e passado no método. Deve ser invocado quando o conteúdo da tab se encontra completamente carregado.

Parâmetro Tipo Default Descrição
tab objeto -- O tab ao qual vai ser aplicado o estado.
status string -- O ícone colocado na tab a representar o seu estado. Pode ser passado qualquer valor desde que haja um match num ficheiro CSS. A FUXI disponibiliza status de success, info, warning, danger.
title string -- A tooltip aplicada ao ícone de estado colocado na tab.
addTab(name, id, dismissible, sortable, selected, customParams, actions, actionsList)

Método que permite adicionar uma nova tab.

Parâmetro Tipo Default Descrição
name string -- Define o título do tab.
id string -- Define o identificador único aplicado ao tab. Este id é aplicado à tag li.
dismissible booleano -- Define se um tab pode ser removido. Caso se defina o parâmetro actions a true o botão de remover tab não será apresentado e deverá ser disponibilizado pelo programador dentro do array das ações passadas no parâmetro actionsList.
sortable booleano -- Define se um tab pode ser reordenado.
selected booleano -- Define se ao adicionar um tab este fica automaticamente selecionado.
customParams objeto -- Permite ao programador passar para dentro da tab informação que ache relevante. No caso de ser necessário passar informação para dentro do template da tab deverá utilizar a propriedade itemTemplParams. ver código
actions booleano -- Define se o tab disponibiliza um conjunto de ações. Se definido a true o botão de remover tab não será apresentado e deverá ser disponibilizado pelo programador dentro do array das ações passadas no parâmetro actionsList.
actionsList array -- Define uma lista com objetos de ações. ver código
selectTabById(tabId)

Método que permite selecionar a tab pelo identificador único presente no li.

removeAllTabs()

Método que permite eliminar todas as tabs.
