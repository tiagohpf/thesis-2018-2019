Accordion - overview

O componente de acordeão é constituido por uma div encapsuladora identificada com as classes panel fx-accordion-navigation, por cabeçalho com a class fx-accordion-navigation-heading e um body com a class fx-accordion-navigation-body.

Dentro do elemento com a class fx-accordion-navigation-body podem ser colocados diferentes tipos de conteúdo. Abaixo encontra-se um exemplo de uma solução em que a área de body possui uma porção de texto.

Importa referir que para que a funcionalidade de acordeão seja garantida, os elementos deverão ter o mesmo data-parent.
