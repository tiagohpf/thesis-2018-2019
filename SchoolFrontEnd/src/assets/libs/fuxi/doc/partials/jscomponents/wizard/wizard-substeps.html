Wizard com substeps

Este snippet demonstra um wizard em que um dos steps é na realidade composto por dois substeps.

Quando a noção de substep é introduzida, o step que alberga os substeps não possui qualquer informação associada. É meramente usado para efeitos de numeração.

Os substeps são definidos através da class fx-nav-link-with-substeps na tag li.

O elemento com a class fx-wizard-wrapper ocupará sempre 100% da altura do seu parent.
