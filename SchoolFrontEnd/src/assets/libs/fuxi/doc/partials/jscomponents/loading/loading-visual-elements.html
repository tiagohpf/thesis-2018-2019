Loading - elementos visuais

A FUXI disponibiliza dois tipos de loaders, cada um em 3 tamanhos distintos. Por defeito os componentes usam loading do tipo spinner.

Dependendo do contexto, pode ser aplicado loading do tipo spinner ou barras. Aconselhamos a utilização de apenas um destes tipos por produto.

No snippet abaixo está exemplificado o tamanho mais pequeno dos loadings spinner e barras . Podem instanciar-se outros tipos de tamanho através das classes fx-ico-loading-m, fx-ico-loading-l ou fx-ico-loading-bars-m, fx-ico-loading-bars-l.

Caso o browser não suporte animações está garantido um fallback para um gif animado.
