Estilização do splitter

Abaixo encontra explicação de algumas classes chave aplicadas ao splitter.

Class Descrição
.fx-uilayout-container Garante que o splitter ocupa sempre a totalidade da altura definida no seu parent.
.fx-uilayout-container-bordered Quando aplicada ao mesmo elemento que contém a class .fx-uilayout-container coloca um border em volta do splitter.
.fx-uilayout-sidebared Deve aplicar-se ao mesmo elemento que contém a class .fx-uilayout-container quando se pretende que uma das sidebars (elemento identificado com a class .fx-sidebar) fique com a cor cinzenta. Se não for definida esta class, a sidebar será branca.
