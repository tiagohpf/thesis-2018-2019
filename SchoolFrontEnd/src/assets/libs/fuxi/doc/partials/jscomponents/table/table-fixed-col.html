Colunas fixas

O DataTables possui um extra que disponibiliza uma forma de fixar colunas da tabela. Em DataTables Fixed Cols pode encontrar informação detalhada.

O Basepack disponibiliza este extra. Para isso deverá incluir o ficheiro FixedColumns.min.js disponível em src="/basepack/datatables-pt/1.9/js/FixedColumns.min.js".

Deverá fazer inclusão dos ficheiros necessários à instanciação de um objeto como DataTable, anteriormente referidos no ponto tabela - overview.

Tenha por favor atenção que o plugin empacotado no basepack apenas funciona até à versão 1.8 do jQuery. Essa é a razão pela qual não apresentamos snippet. Caso necessite de usar uma versão superior do jQuery deverá fazer upgrade para a versão 1.10 do DataTables.
