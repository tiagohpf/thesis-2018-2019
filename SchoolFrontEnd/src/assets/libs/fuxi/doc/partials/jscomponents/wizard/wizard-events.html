Eventos e Métodos

O wizard disponibiliza os eventos e métodos abaixo enumerados.

Eventos

Todos os eventos deverão receber o objeto wizard ao qual dizem respeito. Exemplo: $(document).on('goto.wizard.fuxi', '#wizardID', function (e) {});.

Evento Descrição
goto.wizard.fuxi Executado ao clicar numa das opções desbloqueadas na zona de listagem dos steps.
next.wizard.fuxi Executado quando se avança no wizard através do botão seguinte.
prev.wizard.fuxi Executado quando se retrocede no wizard através do botão anterior.
complete.wizard.fuxi Executado quando se completa o wizard através do botão finalizar.

Métodos

FUXI.Wizard.init({options})

Método que inicializa o wizard e no qual deverão ser passadas as opções pretendidas.

FUXI.Wizard.toggleDynamicSteps($this)

Método que permite mostrar um step dinâmico.

FUXI.Wizard.rearrangeNumeration($this.closest("[data-wizard]"))

Método que permite reordenar a numeração dos steps caso algum step dinâmico tenha sido despoletado.

e.Wizard.setSummary(e.currentStep, yourSummaryFunction[e.currentStep])

Método que permite colocar um sumário das opções escolhidas num step..
