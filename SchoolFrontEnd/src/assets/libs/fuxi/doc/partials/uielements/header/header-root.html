Header

O elemento DOM header especifica o cabeçalho de um documento ou de uma secção.

O seguinte snippet de código ilustra um header de sistema que é composto por diversos elementos (apenas ilustrativos) como:

  • Logótipo e nome do sistema
  • Mudança de sistema
  • Menus de navegação. Ver #docs-menus-simples.
  • Pesquisa transversal
  • Notificações. Ver #docs-menus-dimensoes-2-niveis.
  • Mensagens. Ver #docs-menus-simples.
  • Menu de utilizador Ver #docs-menus-simples.
