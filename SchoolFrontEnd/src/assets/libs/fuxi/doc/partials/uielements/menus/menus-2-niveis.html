Menu com 2 níveis

Este snippet demonstra como utilizar um menu simples com 2 níveis.

Para os níveis estarem disponíveis é necessário aplicar à tag nav as classes carousel e slide. O segundo nível é composto por uma div com a class carousel-inner.

Os itens com um nível abaixo devem ter como indicação uma seta à direita com a class fa-angle-right e fx-slide-indicator.
