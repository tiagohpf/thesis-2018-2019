Botões disabled

Este snippet demonstra como instanciar um botão default no estado disabled. Para qualquer outro tipo de botão devemos substituir o -default pelo tipo que desejarmos (o texto de cada botão representa o tipo de botões admitidos). Podemos também controlar o tamanho dos botões. Para isso ver exemplos acima.

Apesar de os botões ficarem com o aspeto disabled nem todos os browsers vão realmente desabilitar os botões. Por segurança deve-se usar JavaScript para desabilitar completamente os botões.
