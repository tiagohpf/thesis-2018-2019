Como usar

Este snippet demonstra como utilizar um ícone de qualquer uma das icon fonts apresentadas. De acordo com o ícone pretendido a iconclass deve ser substituída pela respetiva classe do ícone. Essa classe encontra-se descrita nos links de detalhe de cada fonte.

Quando a iconografia pretende transmitir algum tipo de significado, deve-se sempre fornecer conteúdo alternativo para que tecnologias assistivas o consigam interpretar. Recomenda-se o uso de conteúdo visual escondido pela classe .sr-only ou o uso do atributo title.
