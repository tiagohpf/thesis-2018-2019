Grupos em formulários

Tipicamente os formulários das aplicações que usam FUXI como base são bastante complexos. Como tal desenvolveu-se uma estratégia que permite ter grupos agrupadores, que podem ser colapsáveis caso tal se pretenda.

Invocando esta porção de código FUXI.ExpandableForm.init(); tornará os grupos colapsáveis.
