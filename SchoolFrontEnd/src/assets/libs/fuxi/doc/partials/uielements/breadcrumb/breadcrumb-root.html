Breadcrumbs

O breadcrumb representado nesta página faz override a alguns estilos da componente bootstrap breadcrumb.

A FUXI recomenda que a primeira migalha do breadcrumb seja precedida do ícone fuxicons-home. Os separadores entre as migalhas são colocados via CSS, não havendo necessidade de serem inseridos no markup.
