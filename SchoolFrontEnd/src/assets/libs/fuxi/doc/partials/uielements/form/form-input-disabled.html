Inputs disabled

Para dar uma aparência de desabilitado aos diferentes inputs basta adicionar o atributo disabled ao input.

Recomendamos usar o atributo title para indicar o motivo pelo qual determinado input está desabilitado.

Por vezes são utilizadas componentes de terceiros que também possuem estado de disabled. Esse aspeto é abordado em cada uma das componentes e não nesta área.
