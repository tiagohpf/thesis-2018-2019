Especificidade

A especificidade em CSS é utilizada para determinar quais as regras que têm precedência (são mais específicas) para estilizar um seletor.

Esta área pretende ser uma zona de consulta rápida e tem como objetivo ajudar a compreender porque determinados estilos declarados têm primasia em relação a outros.
